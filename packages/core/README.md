# @proton-ui/core

Proton UI 核心组件库 - 为 AI 原生场景设计的 React 组件库。

## 安装

```bash
pnpm add @proton-ui/core
```

## 使用

```tsx
import { Button } from "@proton-ui/core";

function App() {
  return <Button variant="primary">点击我</Button>;
}
```

## 开发

```bash
# 开发模式
pnpm dev

# 构建
pnpm build

# 类型检查
pnpm type-check

# 代码检查
pnpm lint
```

## 组件列表

- Button - 按钮组件
- Input - 输入框组件
- Card - 卡片组件
