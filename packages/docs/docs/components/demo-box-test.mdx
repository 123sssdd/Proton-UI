## 总结

DemoBox 组件提供了以下功能：

1. ✅ **标题和描述**：清晰地说明演示内容
2. ✅ **代码展示**：可折叠的代码块，支持语法高亮和复制
3. ✅ **属性面板**：交互式控制组件属性
4. ✅ **多种控制器类型**：
   - Boolean（复选框）
   - String（文本输入）
   - Number（滑块）
   - Select（下拉选择）
   - Color（颜色选择器）
5. ✅ **背景样式**：default、dark、light、transparent
6. ✅ **响应式设计**：适配移动端和桌面端
7. ✅ **样式隔离**：使用 CSS Modules 避免样式冲突

这个组件将成为 Proton UI 文档系统的核心，为所有组件提供统一、专业的演示体验。
