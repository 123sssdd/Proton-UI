# ChatContainer 对话容器

用于展示完整的对话流，内置自动滚动、加载状态和像素风格支持。

## 基础用法

<div className="h-[400px] border rounded-lg overflow-hidden relative">
  <ChatContainer
    messages={[
      { id: "1", role: "assistant", content: "你好！我是 Proton UI AI。" },
      { id: "2", role: "user", content: "能帮我生成一个按钮吗？" },
    ]}
  />
</div>

```tsx
import { ChatContainer } from "@proton-ui/components";

const messages = [
  { id: "1", role: "assistant", content: "你好！" },
  { id: "2", role: "user", content: "Hello!" },
];

<ChatContainer messages={messages} />;
```

## 像素风格容器

为整个聊天窗口应用统一的像素主题。

### Retro Futurism

<div className="h-[400px] overflow-hidden relative">
  <ChatContainer
    pixelTheme="retro-futurism"
    decoration="scanline"
    messages={[
      { id: "1", role: "system", content: "SYSTEM INITIALIZED" },
      { id: "2", role: "assistant", content: "Ready for input." },
      { id: "3", role: "user", content: "Execute protocol 7." },
    ]}
  />
</div>

```tsx
<ChatContainer
  pixelTheme="retro-futurism"
  decoration="scanline"
  messages={messages}
/>
```

## API

| 属性       | 说明       | 类型            | 默认值  |
| ---------- | ---------- | --------------- | ------- |
| messages   | 消息列表   | `ChatMessage[]` | `[]`    |
| loading    | 是否加载中 | `boolean`       | `false` |
| pixelTheme | 像素主题   | `PixelTheme`    | -       |
| decoration | 装饰效果   | `Decoration`    | -       |
