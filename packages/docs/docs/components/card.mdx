# Card 卡片

卡片组件用于展示内容和操作，提供像素风格的视觉效果和灵活的组合模式。

## 基础用法

Card 组件提供简洁的容器样式，可以通过 `padding` 和 `shadow` 属性调整外观。

### 卡片间距

在网格布局中使用适当的间距，确保卡片之间有足够的呼吸空间：

<Callout type="tip" title="间距建议">
  - 使用 `gap-4` (16px) 作为默认间距 - 在紧凑布局中使用 `gap-2` (8px) -
  在宽松布局中使用 `gap-6` (24px) 或 `gap-8` (32px)
</Callout>

<Callout type="warning" title="避免过度嵌套">
  过度嵌套卡片会导致视觉层次混乱。建议最多嵌套 2 层，超过 2
  层时考虑重新设计布局。
</Callout>

## 设计令牌

Card 组件使用以下 CSS 变量，可以通过主题系统进行自定义：

- `--color-bg-primary`: 卡片背景色
- `--color-border`: 卡片边框颜色
- `--color-text-primary`: 标题文本颜色
- `--color-text-secondary`: 副标题和辅助文本颜色
