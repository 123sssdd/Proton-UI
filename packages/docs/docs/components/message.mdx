# Message 消息组件

用于显示对话消息，支持 Markdown 渲染、流式打字机效果和多种像素风格主题。

## 基础用法

<div className="p-4 border rounded-lg bg-white dark:bg-gray-900">
  <Message
    role="assistant"
    content="你好！我是 Proton UI 的 AI 助手。"
    avatar="https://api.dicebear.com/7.x/bottts/svg?seed=proton"
    timestamp={new Date()}
  />
  <Message
    role="user"
    content="请介绍一下像素风格组件。"
    avatar="https://api.dicebear.com/7.x/avataaars/svg?seed=user"
    timestamp={new Date()}
  />
</div>

```tsx
import { Message } from '@proton-ui/components';

<Message
  role="assistant"
  content="你好！我是 Proton UI 的 AI 助手。"
  avatar="url/to/avatar"
  timestamp={new Date()}
/>
<Message
  role="user"
  content="请介绍一下像素风格组件。"
  timestamp={new Date()}
/>
```

## 像素风格主题

Message 组件支持多种精美的像素艺术主题，只需设置 `pixelTheme` 属性。

### Retro Futurism (复古未来)

<div className="p-4 border rounded-lg bg-[#161B33]">
  <Message
    pixelTheme="retro-futurism"
    role="assistant"
    content="系统已启动。等待指令..."
    decoration="scanline"
  />
  <Message pixelTheme="retro-futurism" role="user" content="启动核心引擎。" />
</div>

```tsx
<Message
  pixelTheme="retro-futurism"
  role="assistant"
  content="..."
  decoration="scanline"
/>
```

### Neo Tokyo (新东京)

<div className="p-4 border rounded-lg bg-[#0a0a1a]">
  <Message
    pixelTheme="neo-tokyo"
    role="assistant"
    content="欢迎来到新东京网络层。"
    decoration="glitch"
  />
</div>

### Cyber Shrine (赛博神社)

<div className="p-4 border rounded-lg bg-[#1B4D5C]">
  <Message
    pixelTheme="cyber-shrine"
    role="assistant"
    content="祈福已接收。吉。"
    decoration="rivets"
  />
</div>

## API

| 属性       | 说明         | 类型                                     | 默认值  |
| ---------- | ------------ | ---------------------------------------- | ------- |
| role       | 消息角色     | `'user' \| 'assistant' \| 'system'`      | -       |
| content    | 消息内容     | `string`                                 | -       |
| pixelTheme | 像素主题     | `'retro-futurism' \| 'neo-tokyo' \| ...` | -       |
| decoration | 装饰效果     | `'scanline' \| 'rivets' \| 'glitch'`     | -       |
| streaming  | 是否流式渲染 | `boolean`                                | `false` |
