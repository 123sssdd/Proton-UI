# Changesets

This directory contains changeset files that describe changes to packages in this monorepo.

## Usage

To create a new changeset:

```bash
pnpm changeset
```

To version packages based on changesets:

```bash
pnpm version-packages
```

To publish packages:

```bash
pnpm release
```
